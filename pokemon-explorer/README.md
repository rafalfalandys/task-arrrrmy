# Pokemon Explorer

install: npm i  
run: npm run dev
build: npm run build
build dev: npm run build-dev

## Prologue

Pokemon Explorer is a browser of these little creatures. It uses the 'https://pokeapi.co/' to browse the Pokemons. It displays them basing on given type, and typed name.

## The core

The application first fetch available Pokemon types, then fetch the names belonging to these types, and fetch the details of each creature. All this data is being displayed as a list.

Tech stack: Vue | Typescript | Pinia | primevue components
