# Pokemon Explorer

install: npm i  
run: npm run dev  
build: npm run build  
build dev: npm run build-dev  

## Prologue

Pokemon Explorer is a browser of these little creatures. It uses the 'https://pokeapi.co/' to display the Pokemons and their details. It lists them basing on given type, and typed name.

## The core

The application first fetch available Pokemon types, then fetch the names belonging to these types, and fetch the details of each creature. All this data is being displayed as a list.

Once card is clicked the modal window reveals the details - evelutions, locations and gallery with more pictures

## The core

Tech stack: Vue | Typescript | Pinia | Primevue components
